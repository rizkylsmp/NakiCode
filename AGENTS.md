# Naki Code Agent Instructions

Sebelum mengerjakan task apa pun di repository ini:

1. Baca `docs/PROJECT_SUMMARY.md` secara lengkap sebagai sumber konteks utama project.
2. Ikuti arsitektur, istilah produk, aturan UI, dan workflow yang tertulis di dalamnya.
3. Gunakan `rtk` sebagai prefix untuk setiap operasi shell.
4. Pertahankan perubahan user yang sudah ada dan jangan mengubah file di luar scope task tanpa alasan yang diperlukan.
5. Setelah task selesai, jalankan validasi yang relevan dan tambahkan catatan task ke `docs/CHANGELOG.md`.

Jika instruksi bertentangan, ikuti instruksi dengan prioritas lebih tinggi dan gunakan `docs/PROJECT_SUMMARY.md` sebagai referensi khusus repository.
